# Page And Structure Attributes

- `lang`
  Example: `<html lang="en">`
  Meaning: tells the browser the language of the page.
  Cái này báo browser biết trang này xài ngôn ngữ gì.
  Note: `lang="en"` means the page is in English.
  `lang="en"` là trang này tiếng Anh.

- `charset`
  Example: `<meta charset="UTF-8">`
  Meaning: tells the browser which text encoding to use.
  Cái này nói browser phải đọc chữ theo kiểu nào.
  Note: `UTF-8` is the normal modern choice.
  `UTF-8` là loại thường xài nhất bây giờ.

- `name`
  Example: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
  Meaning: gives the meta tag a specific purpose.
  Cái này đặt tên chức năng cho meta.
  Note: in this project, `name="viewport"` helps mobile display.
  Trong bài này `viewport` giúp hiển thị trên điện thoại đẹp hơn.

- `content`
  Example: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
  Meaning: gives the actual value or instruction for the meta tag.
  Cái này đưa lệnh thiết lập thật sự.
  Note: `content` holds the settings.
  `content` là nơi chứa giá trị cài đặt.

- `http-equiv`
  Example: `<meta http-equiv="refresh" content="0; url=auth/login.html">`
  Meaning: simulates an HTTP instruction inside HTML.
  Cái này giả như lệnh HTTP ngay trong HTML.
  Note: `refresh` is often used for redirecting.
  `refresh` thường dùng để chuyển trang.

- `class`
  Example: `<main class="login-page">`
  Meaning: gives an element a reusable styling name.
  Cái này đặt tên để CSS xài lại nhiều lần.
  Note: CSS uses the class name to style matching elements.
  CSS thấy tên class này thì nó style cho đúng cái đó.

- `title`
  Example: `<title>Login</title>`
  Meaning: sets the text shown in the browser tab.
  Cái này hiện chữ trên tab browser.
  Note: this is the page name shown on the tab.
  Đây là tên trang người ta thấy trên tab.
